InitializeLighter Function

Initializes the properties of a lighter item.

include "Foundation.md";

function InitializeLighter(
    $item : Item,
    $adjectives : String,
    $noun : String,
    $loc : Item
    );

Parameters

$item

Item to initialize.

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.

Source

function InitializeLighter($item:Item, $adjectives:String, $noun:String, $loc:Item)
{
    InitializePortableItem($item, $adjectives, $noun, $loc);
    $item.UseOnAction = UseLighterOn;
}
