NewDoorItem Function

Creates and initializes a door item.

include "Foundation.md";

function NewDoorItem(
    $from : Item,
    $to : Item,
    $dir : Direction,
    $adjectives : String,
    $noun : String,
    $state : DoorState
    ) : Item;

Parameters

$from

Source room to link from.

$to

Destination room to link to.

$dir

Direction of the link.

$adjectives

Space-separated adjectives used to refer to the door.

$noun

Noun used to refer to the door.

$state

Initial state of the door.

Return value

Returns the newly-created door item.

Source

function NewDoorItem($from:Item, $to:Item, $dir:Direction, $adjectives:String, $noun:String, $state:DoorState) : Item
{
    $return = NewItem($"_{$noun}_{$from}{$to}");
    InitializeDoor($return, $from, $to, $dir, $adjectives, $noun, $state);
}
