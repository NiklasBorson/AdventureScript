NewArmor Function

Creates and initializes an armor item.

include "Foundation-Combat.md";

function NewArmor(
    $adjectives : String,
    $noun : String,
    $kind : ArmorKind,
    $damageResistance : Int,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$kind

Value of the ArmorKind property.

$damageResistance

DamanageResistance conferred by the armor.

$loc

Initial location of the item, such as a room or container.

Return value

Returns the newly-created item.

Source

function NewArmor($adjectives:String, $noun:String, $kind:ArmorKind, $damageResistance:Int, $loc:Item) : Item
{
    $return = NewItem($"_{$loc}_{$noun}");
    InitializeArmor($return, $adjectives, $noun, $kind, $damageResistance, $loc);
}
