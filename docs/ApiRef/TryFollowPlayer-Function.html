TryFollowPlayer Function

Helper function that may be used to implement the UpdateAction for monsters and non-player characters. The function moves the monster to the player's location if the monster can follow the player.

include "Foundation-Combat.md";

function TryFollowPlayer(
    $monster : Item
    ) : Bool;

Parameters

$monster

Monster or non-player character.

Return value

Returns true if the monster moved, or false if not.

Source

function TryFollowPlayer($monster:Item) : Bool
{
    if (!$isNowDark && $monster.Location == $lastRoom && $lastRoom != player.Location)
    {
        $monster.Location = player.Location;
        AddItemWords($monster);
        Message($"The {Label($monster)} follows you.");
        $return = true;
    }
}
