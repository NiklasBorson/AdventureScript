InitializeBed Function

Initializes the properties of a bed item.

include "Foundation-Combat.md";

function InitializeBed(
    $item : Item,
    $adjectives : String,
    $noun : String,
    $loc : Item
    );

Parameters

$item

Item to initialize.

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room.

Source

function InitializeBed($item:Item, $adjectives:String, $noun:String, $loc:Item)
{
    InitializeTable($item, $adjectives, $noun, $loc);
    $item.UseAction = UseBed;
}
