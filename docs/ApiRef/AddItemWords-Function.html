AddItemWords Function

Called by InitializeWordMap to add nouns and adjectives for the specified item. This enables the player to refer to the item via commands.

include "Foundation.md";

function AddItemWords(
    $item : Item
    );

Parameters

$item

Item to add nouns and adjectives for.

Source

function AddItemWords($item:Item)
{
    AddAdjectives($item.Adjectives, $item);
    AddNoun($item.Noun, $item);
}
