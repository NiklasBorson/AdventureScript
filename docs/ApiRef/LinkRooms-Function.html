LinkRooms Function

Creates a two-way link between rooms, that is, two one-way links in opposite directions.

include "Foundation.md";

function LinkRooms(
    $from : Item,
    $to : Item,
    $via : Item,
    $dir : Direction
    );

Parameters

$from

Source room to link from.

$to

Destination room to link to.

$via

Intermediate item such as a door, opening, or path.

$dir

Direction of the link.

Source

function LinkRooms($from:Item, $to:Item, $via:Item, $dir:Direction)
{
    LinkRoomsOneWay($from, $to, $via, $dir);
    LinkRoomsOneWay($to, $from, $via, Opposite($dir));
}
