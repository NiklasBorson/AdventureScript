DirectionPhrase Function

Returns a phrase representing a direction.

include "Foundation.md";

function DirectionPhrase(
    $fromValue : Direction
    ) : String;

Parameters

$fromValue

Input direction.

Return value

Returns a phrase such as "to the North" or "leading upward".

Source

map DirectionPhrase Direction -> String
{
    North -> "to the North",
    South -> "to the South",
    East -> "to the East",
    West -> "to the West",
    Up -> "leading upward",
    Down -> "leading downward"
}
