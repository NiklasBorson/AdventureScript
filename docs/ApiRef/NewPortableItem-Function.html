NewPortableItem Function

Creates and initializes a portable item.

include "Foundation.md";

function NewPortableItem(
    $adjectives : String,
    $noun : String,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.

Return value

Returns the newly-created item.

Source

function NewPortableItem($adjectives:String, $noun:String, $loc:Item) : Item
{
    $return = NewItem($"_{$noun}{$loc}");
    InitializePortableItem($return, $adjectives, $noun, $loc);
}
