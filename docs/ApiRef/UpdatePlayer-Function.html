UpdatePlayer Function

The UpdateAction for the player is to heal one health unit.

    $item : Item

Parameters

$item

Item the action is invoked on.

Source

function UpdatePlayer($item:Item)
{
    if ($item.Health < $item.MaxHealth)
    {
        $item.Health = $item.Health + 1;
    }
}
