SetLabelProperties Function

Sets the adjectives and noun for an item, which together comprise its label.

include "Foundation.md";

function SetLabelProperties(
    $item : Item,
    $adjectives : String,
    $noun : String
    );

Parameters

$item

Item to set properties of.

$adjectives

New value of the Adjectives property.

$noun

New value of the Noun property.

Source

function SetLabelProperties($item:Item, $adjectives:String, $noun:String)
{
    $item.Adjectives = $adjectives;
    $item.Noun = $noun;
}
