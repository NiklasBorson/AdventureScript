DrawRectangle Function

Called after BeginDrawing to draw a rectangle.

function DrawRectangle(
    $left : Int,
    $top : Int,
    $width : Int,
    $height : Int,
    $fillColor : Int,
    $strokeColor : Int,
    $strokeThickness : Int
    );

Parameters

$left

Left coordinate of the rectangle.

$top

Top coordinate of the rectangle.

$width

Width of the rectangle.

$height

Height of the rectangle.

$fillColor

ARGB fill color.

$strokeColor

ARGB border color.

$strokeThickness

Border thickness.
