Parameters

$item

$health

Initial value of the Health and MaxHealth properties.

Source

function SetItemHealth($item:Item, $health:Int)
{
    $item.Health = $health;
    $item.MaxHealth = $health;
    $item.DescribeHealthAction = DescribeItemHealth;
}
