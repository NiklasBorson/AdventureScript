NewBed Function

Creates and initializes a bed item.

include "Foundation-Combat.md";

function NewBed(
    $adjectives : String,
    $noun : String,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room.

Return value

Returns the newly-created item.

Source

function NewBed($adjectives:String, $noun:String, $loc:Item) : Item
{
    $return = NewItem($"{$noun}_{$loc}");
    InitializeBed($return, $adjectives, $noun, $loc);
}
