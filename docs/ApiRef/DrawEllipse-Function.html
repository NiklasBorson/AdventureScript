DrawEllipse Function

Called after BeginDrawing to draw an ellipse.

function DrawEllipse(
    $left : Int,
    $top : Int,
    $width : Int,
    $height : Int,
    $fillColor : Int,
    $strokeColor : Int,
    $strokeThickness : Int
    );

Parameters

$left

Left coordinate of the ellipse.

$top

Top coordinate of the ellipse.

$width

Width of the ellipse.

$height

Height of the ellipse.

$fillColor

ARGB fill color.

$strokeColor

ARGB border color.

$strokeThickness

Border thickness.
