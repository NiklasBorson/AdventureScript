SetPlayerDamageResistance Function

Sets the player's DamageResistance to a weighted average of the damage resistence for each piece of armor.

include "Foundation-Combat.md";

function SetPlayerDamageResistance()
{
    player.DamageResistance = ($headArmor.DamageResistance * 30 + $torsoArmor.DamageResistance * 50 + $legArmor.DamageResistance * 20) / 100;
}
