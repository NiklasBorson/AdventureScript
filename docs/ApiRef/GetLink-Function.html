GetLink Function

Gets the linked item, if any, for the specified item in the specified direction.

include "Foundation.md";

function GetLink(
    $item : Item,
    $dir : Direction
    ) : Item;

Parameters

$item

Item to get the linked item for.

$dir

Direction of the link.

Return value

Returns the linked item in the specified direction, if any.

Source

function GetLink($item:Item, $dir:Direction) : Item
{
    switch ($dir)
    {
        case Direction.North
        {
            return $item.LinkN;
        }
        case Direction.South
        {
            return $item.LinkS;
        }
        case Direction.East
        {
            return $item.LinkE;
        }
        case Direction.West
        {
            return $item.LinkW;
        }
        case Direction.Up
        {
            return $item.LinkU;
        }
        case Direction.Down
        {
            return $item.LinkD;
        }
    }
}
