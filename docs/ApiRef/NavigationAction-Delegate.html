NavigationAction Delegate

Delegate type for the LeaveAction and EnterAction properties.

include "Foundation.md";

delegate NavigationAction(
    $from : Item,
    $to : Item
    ) : Bool;

Parameters

$from

Room being left.

$to

Room being entered.

Return value

Return true to cancel navigation, or false to allow it.
