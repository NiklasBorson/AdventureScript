NewLighter Function

Creates and initializes a lighter item.

include "Foundation.md";

function NewLighter(
    $adjectives : String,
    $noun : String,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.

Return value

Returns the newly-created item.

Source

function NewLighter($adjectives:String, $noun:String, $loc:Item) : Item
{
    var $item = NewItem($"_{$noun}_{$loc}");
    InitializeLighter($item, $adjectives, $noun, $loc);
    return $item;
}
