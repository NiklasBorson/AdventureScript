AdventureScript

NewLink Function

Creates and initializes an unnamed link between rooms. The link object does not have a noun so is not automatically enumerated by the Describe function. The room's Description property may therefore need to explicitly mention the link, e.g., "You are in a clearing with a path to the north."

include "Foundation.md";

function NewLink(
    $from : Item,
    $to : Item,
    $dir : Direction
    ) : Item;

Parameters

$from

Source room to link from.

$to

Destination room to link to.

$dir

Direction of the link.

Return value

Returns the newly-created unnamed link item.

Source

function NewLink($from:Item, $to:Item, $dir:Direction) : Item
{
    return NewDoorItem($from, $to, $dir, "", "", DoorState.None);
}
