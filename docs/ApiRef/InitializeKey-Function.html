InitializeKey Function

Initializes the properties of a key item.

include "Foundation.md";

function InitializeKey(
    $key : Item,
    $adjectives : String,
    $noun : String,
    $loc : Item
    );

Parameters

$key

Item to initialize.

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.

Source

function InitializeKey($key:Item, $adjectives:String, $noun:String, $loc:Item)
{
    InitializePortableItem($key, $adjectives, $noun, $loc);
    $key.UseOnAction = UseKeyOn;
}
