IsOpenContainer Function

Tests whether an item is an open container.

function IsOpenContainer(
    $item : Item
    ) : Bool;

Parameters

$item

Item to test.

Return value

Returns true if the item is a container which is not closed or locked.

Source

function IsOpenContainer($item:Item) => IsContainer($item) && !IsClosedOrLocked($item.DoorState);
