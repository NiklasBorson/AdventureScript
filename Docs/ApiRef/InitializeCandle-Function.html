InitializeCandle Function

Initializes the properties of a light source that behaves like a candle.

function InitializeCandle(
    $item : Item,
    $adjectives : String,
    $noun : String,
    $loc : Item
    );

Parameters

$item

Item to initialize.

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.
