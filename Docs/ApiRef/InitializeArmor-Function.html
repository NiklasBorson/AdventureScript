InitializeArmor Function

Initializes the properties of an armor item.

function InitializeArmor(
    $item : Item,
    $adjectives : String,
    $noun : String,
    $kind : ArmorKind,
    $damageResistance : Int,
    $loc : Item
    );

Parameters

$item

Item to initialize.

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$kind

Value of the ArmorKind property.

$damageResistance

DamanageResistance conferred by the armor.

$loc

Initial location of the item, such as a room or container.
