NewTable Function

Creates and initializes a table item.

function NewTable(
    $adjectives : String,
    $noun : String,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room.

Return value

Returns the newly-created table item.

Source

function NewTable($adjectives:String, $noun:String, $loc:Item) : Item
{
    $return = NewItem($"_{$noun}_{$loc}");
    InitializeTable($return, $adjectives, $noun, $loc);
}
