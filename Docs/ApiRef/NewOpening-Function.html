NewOpening Function

Creates and initializes an "opening" between rooms.

function NewOpening(
    $from : Item,
    $to : Item,
    $dir : Direction
    ) : Item;

Parameters

$from

Source room to link from.

$to

Destination room to link to.

$dir

Direction of the link.

Return value

Returns the newly-created opening item.

Source

function NewOpening($from:Item, $to:Item, $dir:Direction) : Item
{
    return NewDoorItem($from, $to, $dir, "", "opening", DoorState.None);
}
