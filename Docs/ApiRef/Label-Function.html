Label Function

Gets an item's label, which is a string that can be used to refer to the item.

function Label(
    $item : Item
    ) : String;

Parameters

$item

Item to get the label of.

Return value

Returns the item label.

Source

function Label($item:Item) => $item.Adjectives != null ? $"{$item.Adjectives} {$item.Noun}" : $item.Noun;
