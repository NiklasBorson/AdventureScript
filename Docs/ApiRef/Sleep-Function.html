Sleep Function

Causes the player to sleep for the specified number of minutes or until the sleep is interrupted (i.e., if the player is attacked).

function Sleep(
    $minutes : Int
    ) : Bool;

Parameters

$minutes

Number of minutes to sleep.

Return value

Returns true if the player slept for the full duration, or false if the sleep was interrupted.

Source

function Sleep($minutes:Int) : Bool
{
    $isSleeping = true;
    while ($isSleeping && $minutes > 0)
    {
        Tick();
        if (!$isSleeping)
        {
            return false;
        }
        $minutes = $minutes - 1;
    }
    $isSleeping = false;
    return true;
}
