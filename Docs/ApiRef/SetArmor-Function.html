SetArmor Function

Set the current armor of the specified kind.

function SetArmor(
    $kind : ArmorKind,
    $item : Item
    );

Parameters

$kind

Kind of armor to set.

$item

Item to set, which can be null.

Source

function SetArmor($kind:ArmorKind, $item:Item)
{
    switch ($kind)
    {
        case ArmorKind.Head
        {
            $headArmor = $item;
        }
        case ArmorKind.Torso
        {
            $torsoArmor = $item;
        }
        case ArmorKind.Leg
        {
            $legArmor = $item;
        }
    }
}
