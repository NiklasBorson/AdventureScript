NewLockedDoor Function

Creates and initializes a door item in the Locked state.

function NewLockedDoor(
    $from : Item,
    $to : Item,
    $dir : Direction,
    $key : Item
    ) : Item;

Parameters

$from

Source room to link from.

$to

Destination room to link to.

$dir

Direction of the link.

$key

Item used to unlock the door (see NewKey or InitializeKey).

Return value

Returns the newly-created door item.
