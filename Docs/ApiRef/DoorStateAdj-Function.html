DoorStateAdj Function

Converts a DoorState value to an adjective.

function DoorStateAdj(
    $fromValue : DoorState
    ) : String;

Source

map DoorStateAdj DoorState -> String
{
    None -> "",
    Open -> "open",
    Closed -> "closed",
    Locked -> "closed"
}
