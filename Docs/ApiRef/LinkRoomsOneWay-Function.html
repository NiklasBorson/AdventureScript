LinkRoomsOneWay Function

Creates a one-way link from one room to another.

    $from : Item,
    $to : Item,
    $via : Item,
    $dir : Direction
    );

Parameters

$from

Source room to link from.

$to

Destination room to link to.

$via

Intermediate item such as a door, opening, or path.

$dir

Direction of the link.
