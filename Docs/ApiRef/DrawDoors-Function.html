DrawDoors Function

Draws doors and openings for the specified room.

function DrawDoors(
    $room : Item,
    $left : Int,
    $top : Int,
    $width : Int,
    $height : Int,
    $backColor : Int
    );

Parameters

$room

Room to draw doors and openings for.

$left

Left coordinate of the room.

$top

Top coordinate of the room.

$width

Width of the room.

$height

Height of the room.

$backColor

Background (floor) color of the room.
