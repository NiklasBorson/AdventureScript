InvokeItemAction Function

Invokes an ItemAction delegate or displays an error message.

function InvokeItemAction(
    $func : ItemDelegate,
    $item : Item,
    $verb : String
    );

Parameters

$func

Delegate to invoke, which is typically a property of $item.

$item

Item passed as a parameter to the delegate.

$verb

Verb used in an error message if the delegate is null.
