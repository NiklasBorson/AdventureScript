InvokeItemActionWithFallback Function

Invokes an ItemAction delegate or invokes a fallback action.

function InvokeItemActionWithFallback(
    $func : ItemDelegate,
    $item : Item,
    $fallback : ItemDelegate
    );

Parameters

$func

Delegate to invoke, which is typically a property of $item.

$item

Item passed as a parameter to the delegate.

$fallback

Alternative delegate to invoke if $func is null.

Source

function InvokeItemActionWithFallback($func:ItemDelegate, $item:Item, $fallback:ItemDelegate)
{
    if ($func != null)
    {
        $func($item);
    }
    else
    {
        $fallback($item);
    }
}
