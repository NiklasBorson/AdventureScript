NewContainer Function

Creates and initializes a container item.

function NewContainer(
    $adjectives : String,
    $noun : String,
    $state : DoorState,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$state

Initial state of the container, such as Closed or Locked.

$loc

Initial location of the item, such as a room.

Return value

Returns the newly-created container item.

Source

function NewContainer($adjectives:String, $noun:String, $state:DoorState, $loc:Item) : Item
{
    var $container = NewItem($"_{$noun}_{$loc}");
    InitializeContainer($container, $adjectives, $noun, $state, $loc);
    return $container;
}
