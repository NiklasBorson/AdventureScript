InitializeDoor Function

Initializes the properties of a door item.

function InitializeDoor(
    $door : Item,
    $from : Item,
    $to : Item,
    $dir : Direction,
    $adjectives : String,
    $noun : String,
    $state : DoorState
    );

Parameters

$door

Door item to the the properties of.

$from

Source room to link from.

$to

Destination room to link to.

$dir

Direction of the link.

$adjectives

Space-separated adjectives used to refer to the door.

$noun

Noun used to refer to the door.

$state

Initial state of the door.
