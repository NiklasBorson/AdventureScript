NewLight Function

Creates and initializes a light source that behaves like an electric light.

function NewLight(
    $adjectives : String,
    $noun : String,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.

Return value

Returns the newly-created light source item.

Source

function NewLight($adjectives:String, $noun:String, $loc:Item) : Item
{
    var $item = NewItem($"_{$noun}_{$loc}");
    InitializeLight($item, $adjectives, $noun, $loc);
    return $item;
}
