InitializeContainer Function

Initializes the properties of a container item.

function InitializeContainer(
    $container : Item,
    $adjectives : String,
    $noun : String,
    $state : DoorState,
    $loc : Item
    );

Parameters

$container

Item to initialize.

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$state

Initial state of the container, such as Closed or Locked.

$loc

Initial location of the item, such as a room.
