IsDarkAtNight Function

Function that may be assigned to the IsDark property if a room is dark only at night, e.g., if a room is outside or has windows.

function IsDarkAtNight(
    $item : Item
    ) : Bool;

Parameters

$item

Room on which the predicate is invoked.

Return value

Returns true if a light source is required to see in the room.
