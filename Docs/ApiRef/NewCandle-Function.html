NewCandle Function

Creates and initializes a light source that behaves like a candle.

function NewCandle(
    $adjectives : String,
    $noun : String,
    $loc : Item
    ) : Item;

Parameters

$adjectives

Space-separated adjectives used to refer to the item.

$noun

Noun used to refer to the item.

$loc

Initial location of the item, such as a room or container.

Return value

Returns the newly-created light source item.
