AddStateQualifier Function

Adds a qualifier to the specified phrase if applicable based on the state of the item.

function AddStateQualifier(
    $item : Item,
    $phrase : String
    ) : String;

Parameters

$item

Item to check the state of.

$phrase

Input phrase to qualify.

Return value

Returns the qualified phrase, which may equal the input phrase.
